# NASM
 Netwide Assembler (NASM), an asssembler for the x86 CPU architecture. (ASM for Linux)
## Install NASM
- Ubuntu 20.04
- Install
```
sudo apt update
sudo apt-get -y install nasm
```
- Check
```
nasm --version
```
   + Result
   ```
   root@mail:~# nasm --version
   NASM version 2.15.05
   ```
